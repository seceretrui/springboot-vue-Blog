# springboot-vue-Blog
vue + springboot实现的博客系统

# 实现功能



## 整体

- 用户：登录 注册 退出 注销
- 首页：文章列表、最新文章、最热文章
- 文章分类：列表
- 文章：写文章
- 评论：添加评论、回复评论

## 后端

* 用户、文章、文章分类、评论增删改查
* 操作日志记录
